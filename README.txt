Este programa fue realizado con NODE JS version 16.17.1

Para correr el programa, es necesario abrir la linea de comandos en la carpeta donde está el archivo app.js.
Posterior a eso tecleamos lo siguiente:
    <--Node
    Nos dará la bienvenida a NodeJS. Ahora utilizamos .load para correr el programa
    <--.load app.js 

De esta manera se pueden visualizar los resultados.
